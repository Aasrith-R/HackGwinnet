üöÄ How to Test the Mobile App

Due to macOS file watching limitations, here are your testing options:

Option 1: Use Expo Go on Your Phone (Recommended)
1. Install "Expo Go" from App Store/Google Play
2. Run: npx expo start --tunnel
3. Scan QR code with your phone
Option 2: Fix File Limit and Use iOS Simulator
1. Run: sudo launchctl limit maxfiles 65536 200000
2. Run: npx expo start
3. Press 'i' for iOS simulator
Option 3: Use Android Emulator
1. Start Android emulator
2. Run: npx expo start
3. Press 'a' for Android emulator
